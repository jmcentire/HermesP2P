Three Principles

Every design decision flows from these non-negotiable commitments.

Ephemeral by Design

Messages are transient and never persisted. Once received, they exist only in memory. No server stores your history. No database to subpoena. No breach to disclose.

Decentralized Network

No central servers, no authority, no single point of failure. Every node contributes to the network's resilience. Peer discovery builds a diverse, adaptive connection graph.

Key Sovereignty

Your Ed25519 keypair is your identity. Private keys never leave your device. No passwords, no accounts, no third-party authentication. You control who you are.

How It Works

From bootstrap to encrypted messaging in four steps.

1

Bootstrap

Visit a host node to connect to the network. Upload your config file or generate a fresh keypair with default channel subscriptions.

2

Peer Discovery

Your node discovers peers and builds a diverse connection graph. The network self-organizes for resilience — no coordinator needed.

3

Channels & Messages

Join public channels, create private channels with AES-256 symmetric keys, or send direct messages encrypted with recipients' public keys.

4

Message Propagation

Messages propagate through the network with TTL hop counts. Fixed-size padding prevents traffic analysis. Messages are signed and verified at every hop.

Stateless Authentication

There are no accounts. No passwords. No sessions stored on a server.

Your configuration file IS your identity. It contains your Ed25519 keypair, your channel subscriptions, your friends' public keys, and your peer list. Upload it to any node and you're authenticated.

Download it when you leave. The node forgets you existed. This is authentication without state — more secure than any password database.

Config File
  |
  +-- keypair: Ed25519 public/private
  +-- channels: subscribed topics
  +-- friends: public keys of contacts
  +-- peers: known network nodes
 
> Upload to any node = authenticated
> Download when done = node forgets you
> No server state. No breach surface.

Get Started
